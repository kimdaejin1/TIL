# Git
## Git 간편 사용법
- 레포지토리 클론 받기  
    ~~~
    git clone 레포지토리 경로
    ~~~

- commit & push
    ~~~
    git add .
    git commit -m "[commit message]"
    git push
    ~~~

- 브랜치 생성
    ~~~
    git checkout -b [branch name]
    ~~~
    주의* 브랜치 명은 업무 개요로 만들자

- 브랜치 이동
    ~~~
    git checkout [branch name]
    ~~~

- pull 받기  
    ~~~
    git pull
    ~~~

- 원격 master로부터 pull
    ~~~
    git pull origin master
    ~~~
