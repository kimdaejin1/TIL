# 전위/후위/중위 표기법
## 수식 표현 법의 종류

1. 전위 표기법
    - 연산자를 먼저 표시하고 연산에 필요한 피연산자를 표기하는 방법  
    ex) +AB
2. 중위 표기법
    - 연산자를 피연산자 사이에 집어넣어 표기하는 방법  
    ex) A+B
3. 후위 표기법
    - 피연산자를 먼저 쓰고 연산자를 뒤에 표기하는 방법  
    ex) AB+

* * * 

## 수식 표기법의 전환

- 중위 -> 전위
    1. 중위 표기식에 우선순위에 맞게 괄호를 친다  
    (3+(2+(4*5)))+(3/1)
    2. 괄호 안에 있는 연산자를 앞으로 이동 시킨다.   
    +(+3(+2(*45))(/31)
    3. 괄호를 없애준다  
    ++3+2*45/31
- 중위 -> 후위
    1. 중위 표기식에 우선순위에 맞게 괄호를 친다.  
    (3+(2+(4*5)))+(3/1)
    2. 괄호 안에 있는 연산자를 뒤로 이동 시킨다.  
    (3(2(45*)+)+)(31/)+
    3. 괄호를 없애준다.
    3245*++31/+

    중위로 돌리는 방법은 반대로 하게 되면 다시 돌릴 수 있다.
